The source code is in lib.py

webpage: https://LoucOoc.github.io

Directories:

  -input: test input images

  -R: reflectance of tested images

  -L: illumination of tested images

  -output: output of the tested images of the Weighted Variational Model

  -simple: output of the tested images of a simple pixel intensity increasing model

  -gamma: output of the tested images with only gamma correction

  -overshoot: interesting result with inapporiate hyper-parameters

Reference:
X. Fu, D. Zeng, Y. Huang, X.-P. Zhang, and X. Ding, “A weighted variational model for simultaneous reflectance and illumination estimation,” in Proceedings of the IEEE Conference on Computer Vision and Pattern Recognition (CVPR), 2016, pp. 2782–2790.
